## Project overview

This repository is the **source** for a public collection of [Agent Skills](https://agentskills.io)-style packages. Each skill is a folder with a `SKILL.md` file. Consumers install via **`npx skills@latest`** or the Bash scripts under `scripts/`. Use this index with brownfield PRD/planning workflows to give an AI agent stable grounding in repo layout and conventions.

## Getting started

1. **Read** [README.md](../README.md) for philosophy and install one-liners.
2. **List skills from GitHub** without installing:  
   `npx skills@latest add danielvm-git/skills --list`
3. **Install everything for Cursor (global):**  
   `npx skills@latest add danielvm-git/skills -g -a cursor -y --skill '*'`
4. **Install specific skills only:**  
   `npx skills@latest add danielvm-git/skills -g -a cursor -y --skill tdd --skill write-a-skill`
5. **Confirm in the agent:** start a new chat in Cursor and mention a skill by name (e.g. “use the tdd skill”).
