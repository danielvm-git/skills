# BMAD Method Reference: Document Lifecycle

**Source:** `BMAD-METHOD`
**Author:** Daniel VM

## Core Concepts
- **Bold, Minimal, Actionable, Durable:** The four criteria for high-integrity documentation.
- **The Lifecycle Arc:** Discover → Elaborate → Plan → Build → Sustain.
- **Spec-First Engineering:** Documentation is the source of truth for all automated processes.
