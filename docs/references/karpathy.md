# Karpathy Reference: Agentic Principles

**Source:** [Andrej Karpathy Skills](https://github.com/multica-ai/andrej-karpathy-skills)
**Author:** Andrej Karpathy (curated by Multica AI)

## Core Concepts
- **Think First:** Explicitly surface assumptions before writing code.
- **Simplicity:** Prioritize the minimum viable implementation.
- **Surgical Changes:** Modify only what is required.
- **Loop until Correct:** Continuous verification until behavioral correctness is achieved.
