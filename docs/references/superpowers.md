# Superpowers Reference: Agentic Workflow

**Source:** [Superpowers](https://github.com/obra/superpowers)
**Author:** Jesse Vincent (obra)

## Core Concepts
- **Workflow Automation:** Streamlining the git-branch-test-review-merge loop for agents.
- **Subagent Delegation:** Dispatching independent agents for isolated tasks or reviews.
- **Skill-Based Architecture:** Organizing agent capabilities into discrete, verb-noun "skills".
