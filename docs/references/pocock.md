# Matt Pocock Reference: Agent Skills

**Source:** [Matt Pocock's Skills](https://github.com/mattpocock/skills/tree/main/skills)
**Author:** Matt Pocock

## Core Concepts
- **Deep Modules:** Favoring depth over breadth in module design.
- **Information Hiding:** Reducing the surface area exposed to agents to minimize cognitive load (and token usage).
- **Ubiquitous Language:** Extracting and maintaining a consistent domain language.
- **Zoom-Out:** Always understanding the broader context before zooming into a specific task.
