# Jaroslaw Wasowski Reference: SDD & BDD

**Source:** [SDD: Writing Specifications for AI — BDD as the Missing Link](https://medium.com/@wasowski.jarek/sdd-writing-specifications-for-ai-bdd-as-the-missing-link-spec-driven-development-ad1b540b7f75)
**Author:** Jaroslaw Wasowski

## Core Concepts
- **Spec-Driven Development (SDD):** Using specifications as the primary driver for AI agent behavior.
- **BDD as the Missing Link:** Leveraging Gherkin-style scenarios to provide unambiguous instructions to agents.
- **Verification Loops:** Closing the loop between specification and implementation via automated testing.
