# Agentic Stack Reference: 7 Tools & 5 Layers

**Source:** [The Agentic Coding Stack: 7 Tools, 5 Layers](https://blog.devgenius.io/the-agentic-coding-stack-7-tools-5-layers-and-the-missing-link-nobody-has-built-yet-de264b260db3)

## Core Concepts
- **Layered Intelligence:** Understanding the hierarchy of tools needed for a complete agentic workflow.
- **Tool Orchestration:** How agents should leverage specialized tools for research, planning, and execution.
- **The "Missing Link":** Addressing the gap between high-level planning and low-level execution.
